
# Fit Life - Static Website (student prototype)

This folder contains a simple 5-page static website prototype for Fit Life.
Files include:
- index.html, about.html, courses.html, pricing.html, contact.html, privacy.html
- style.css, script.js

## To publish on GitHub Pages
1. Create a new public repository on GitHub.
2. Upload all files in this folder to the repository root (use Add file → Upload files).
3. In the repository Settings → Pages, set Source to 'Deploy from a branch' and choose 'main' branch, root folder. Save.
4. Wait a minute. Your site will be available at: https://your-username.github.io/your-repo-name

If you want, you can rename repository to `username.github.io` to serve the site at https://username.github.io/
